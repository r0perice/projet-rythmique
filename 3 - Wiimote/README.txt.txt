==============================================
Fichier uniwii.dll
==============================================
Copier le fichier "uniwii.dll" dans le dossier 
C:\Windows 
et dans le dossier
C:\Windows\System32
==============================================
Dans Unity
==============================================
Pour créer l'éxécutable :
1.File->Build Settings
2.Choisir la plateforme windows (sélectionner
par défaut)
3.Choisir x86 (et non x64_86) dans "Architecture"
4.Cliquer sur "Build And Run"
5.Enjoy ;)
